# ⚡ Local Development

## Clone Repository

```bash
git clone https://github.com/saikirantechy/FanVerse-AI.git
cd FanVerse-AI
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

---

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn api.index:app --reload
```
